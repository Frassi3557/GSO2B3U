1.	Request Verbs

	POST	 - 	Sækir gögn sem eru nú þegar til í gegnum URLið.
	GET 	 - 	Býr til ný  gögn  og skilgreinir þau.
	PUT 	 - 	Uppfærir gögn sem eru nú þegar til.
	DELETE - 	Eyðir gögnum.
	

2.	Status Codes

	200		- 	Beiðni til server heppnaðist.
	301		- 	Búið er að breyat URL / færa vefsíðu annað.
	400		- 	Server náði ekki að lesa beiðni og vinna úr henni.
	404		-		Vefsíða er ekki til á server.
	500		-		Server villa.
	
	
3.	HTTP request/response

	* Finnur IP addressu vefsíðunnar í gegnum DNS.
	* Client myndar tengingu við serverinn.
	* Client sendir beiðni á serverinn.
	* Client bíður eftir svari frá server.
	* Lokar tengingu.
